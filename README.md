# Таблица умножения — Telegram бот

Бот для изучения таблицы умножения с режимами:
- Учёта конкретного числа (2-9)
- Вся таблица (случайные примеры)
- Статистика правильных ответов
- 3 попытки на каждый пример

## Команды
- `/start` — начать обучение
- `/cancel` — завершить сессию

## Локальный запуск

##1. Клонируй репозиторий:
```bash
git clone https://github.com/ваш_username/math_bot.git
cd math_bot
##2. Установи зависимости:

bash
pip install -r requirements.txt
##3. Создай файл .env и добавь токен:

text
TELEGRAM_BOT_TOKEN=123456:ABC-DEF...
##4. Запусти бота:

bash
python bot.py
